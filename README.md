# DTU B.Tech Major Project-II LaTeX Template

A complete, ready-to-use LaTeX template for B.Tech Major Project-II report submission at **Delhi Technological University (DTU)**, fully compliant with official DTU guidelines.

---

| Document | Guideline |
|---|---|

| Grading Policy | F.No. DTU/IRD/002/2019/1455 |
| Submission Requirements | DTU/Exam/2019/O/9 |

---

## 🎓 Grading Policy (Quick Reference)

| Grade | Requirement |
|---|---|
| **O / A+** | Paper accepted/published in SCI/SSCI/Scopus **Journal** |
| **A+ / A** | Paper accepted/published in Scopus-indexed **Conference** |
| **B+ and below** | Viva performance only |

---

## ✨ Features

- ✅ Pre-filled Candidate's Declaration (Annexure-I format)
- ✅ Supervisor Certificate
- ✅ All 7 chapters with `% INSTRUCTION:` comments guiding what to write
- ✅ Publication-ready tables (SOTA, Ablation, Metrics, Implementation, Datasets)
- ✅ All 5 required appendices:
  - Appendix A — List of Publications
  - Appendix B — Research Paper Acceptance Proof
  - Appendix C — Conference Registration Proof
  - Appendix D — Scopus Indexing Proof
  - Appendix E — Plagiarism Report (multi-page support)
- ✅ IEEE-format reference examples
- ✅ Times New Roman 12pt, 1.5 line spacing, correct margins
- ✅ Compiles with **pdfLaTeX** — no XeLaTeX needed

---

## 🚀 Quick Start

### Overleaf
1. Download the ZIP
2. Go to [Overleaf](https://overleaf.com) → **New Project** → **Upload Project**
3. Upload the ZIP
4. Set compiler to **pdfLaTeX**
5. Click **Compile**
